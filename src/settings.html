Sets QBWebUIHelper as the default handler for magnet: links and .torrent files. Before uninstalling, click Restore Previous Default to hand back to your previous app.

One more step required

Windows may have a system-level .torrent or magnet association that overrides ours. Please complete the steps below to ensure QBWebUIHelper is used as the default handler.

  1. Click Open Default Apps below.
  2. Search for .torrent → select QBWebUIHelper.
  3. Search for magnet → select QBWebUIHelper.

If you ever decide to uninstall this app, please click Unregister first to cleanly remove all file associations before uninstalling.

⚠ Before you uninstall QBWebUIHelper

macOS has no automatic uninstaller. If you drag QBWebUIHelper to the Trash without clicking Restore Previous Default first, your previous magnet: and .torrent handler (e.g. Transmission) will not be automatically restored.

Always open Settings → Restore Previous Default before uninstalling.
